# Iconos PWA

- **SVG** (ya incluidos): `icon-192.svg`, `icon-512.svg` — funcionan en la mayoría de navegadores.
- **PNG** (recomendado iOS/Android): añade `icon-192.png` (192×192) e `icon-512.png` (512×512) para mejor compatibilidad. Puedes exportar desde los SVG con cualquier editor o herramienta de conversión.
